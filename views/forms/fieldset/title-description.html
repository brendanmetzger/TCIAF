About Markdown

Markdown is a system for adding simple characters to stylize text as well as create links, embed images, and make lists. You can use the toolbar for basic formatting, but with practice, it's quite easy to just input manually as you type. When editing markdown, if there is any doubt that things are correct, simply save and review the output, and continue to edit should anything appear incorrect.

Headings

Headings are just what they sound like, but unlike bold and italic, they have semantic meaning in that search engines and devices that aid accessibility can derive the correct meaning from them. The also have styles associated, so consistent use will ensure proper colors, margins, and default styles for a consistent look site-wide.

Italics, Bold, Special, Lists and Rules

Italics and bold are straightforward, as seen in the toolbar. If you italicize and bold text (***like so***) it will be a bright color instead of italicized or bolded. Lists can be ordered by prefixing lines with a number, or unorderd (bulleted) by prefixing with a hyphen (-) or asterisk (*). Horizontal rules can be created by a string of three or more hyphens (---) or asterisks (***) on its very own line.

Links and Images

The syntax for adding links and images is very similar because they are conceptually similar. When adding a link (or image), you are providing a reference to another piece of content via a url. While you can do that manually with images, often you will just drag and drop from the media panel and it will write the syntax for you. When adding hyperlinks, there is a button in the toolbar, but there are some important things to keep in mind.

  • urls that start with http:// followed by a domain name will open in a new window.
  • urls that start with a forward slash (/) will take advantage of the persistent player.
  • When linking to another site, the link should be starting with http, ie. [Wbez](http://wbez.org)
  • When linking to your content, the link should start with a slash and not have a domain, ie. [our library](/overview/library)
